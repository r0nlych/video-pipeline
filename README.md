# video-pipeline

A long-term project for building a text-driven video generation pipeline.

## Layers
1. Input Layer
2. Script Layer
3. Decomposition Layer
4. Output Layer

## Current Goal
Build a usable MVP:
text input -> structured JSON -> script -> subtitles

## Repository Structure
- docs/: plans and decisions
- src/: code modules
- schemas/: shared data structure
- examples/: sample inputs and outputs
- outputs/: generated results
- prompts/: prompt templates
- experiments/: temporary experiments
